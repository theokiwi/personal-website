Repositório: simpledb

  • Descrição

    Uma base de dados não sql simples para a matéria "Sistemas Operacionais" na PUC Minas. Feita em C++. Contém multithread e IPC message queue. Nela estão disponíveis as funcionalidades:

    1. Iniciar ./db_main
    2. Inserir: ./client_main simpledb insert key value
    3. Remover: ./client_main simpledb remove value
    4. Atualizar: ./client_main simpledb update oldValue newValue;
    5. Pesquisar: ./client_main simpledb search value
    6. Sair da DB: ./client_main simpledb quit
    7. Sair do Client: ./client_main quit
